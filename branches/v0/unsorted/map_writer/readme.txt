generic map writer for tile based games (Gods, Chaos Engine, ...) by JOTD

compiles under MinGW, Cygwin, whatever. SDL required (.dlls included)

examples:

map_writer  gods_config ../Gods/demo-rip.map godout.bmp
map_writer  mp_config ../magicpockets/mp_data_pc_unpacked/LEVEL1A.PAP level1a.bmp
map_writer  mp_config ../magicpockets/mp_data_pc_unpacked/PCMAP.PAP pcmap.bmp
